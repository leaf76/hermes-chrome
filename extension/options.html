Hermes Chrome

Local companion so agents can operate Chrome without hijacking your active tab. Talks only to a bridge on this machine (default port 19876). No cloud account. No analytics.

The bridge requires a shared token by default. Use Pair with bridge within a few minutes of bridge start, or paste the token from ~/.hermes/run/hermes-chrome/bridge.env.

Must match host_permissions (127.0.0.1 / localhost port 19876 for the published build).
