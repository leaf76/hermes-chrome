# Hermes Chrome

Local companion so Hermes / AI agents can operate **your Chrome** without hijacking the tab you are using.

- **Repository:** [github.com/leaf76/hermes-chrome](https://github.com/leaf76/hermes-chrome)
- **Privacy policy:** [privacy-policy.md](./privacy-policy.md)
- **Chrome Web Store package:** see repo `store/` after `./store/package.sh`

Default agent workspace uses a Chrome Tab Group; the product scope is broader agent-friendly browser control.
